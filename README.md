# Чарівні Поні

Гра про поні для дітей 4–8 років. Працює в будь-якому сучасному браузері — відкрий `index.html`.

## Що всередині
- **Вибір поні** — на початку можна обрати одну із семи оригінальних поні:
  Іскорка (єдиноріг-чарівниця), Веселка (швидка пегаска), Квіточка (добра пегаска),
  Перлинка (модниця-єдиноріг), Промінчик (весела), Опівніч (мрійниця-аліконь),
  Пінкі (святкова, з 10 позами). У кожної свої кольори, грива, мітка на боці.
- **Догляд за поні** — годуємо, миємо, розчісуємо, обіймаємо, кладемо спати. У поні
  є шкали ситості, чистоти, бадьорості та радості. Якщо довго не заходити — поні
  починає сумувати.
- **Розмальовка** — розмальовуємо єдиноріжку фарбами з палітри. Натисни на частину
  поні, щоб залити її кольором.
- **Аркади** (можна обрати будь-яку):
  - **Лови зірочки** — збирай зірочки та яблучка, що падають.
  - **Стрибай по веселці** — раннер зі стрибками через камінці.
  - **Знайди пару** — картки з поні, треба відкривати парами.
- **Уроки доброти** — короткі ситуації з вибором правильного вчинку. Текст і
  варіанти озвучуються українським голосом, поточний рядок підсвічується.

## Прогрес
Увесь прогрес зберігається автоматично у `localStorage`:
- стан поні й час останнього візиту (поки тебе немає — поні трохи зголодніє),
- кольори розмальовки,
- найкращі результати в кожній аркаді,
- статистика з уроків доброти,
- обрана поні.

## Озвучка
- Звукові ефекти синтезуються через Web Audio API — без зовнішніх файлів.
- Голос для уроків доброти бере перший доступний `uk-UA` голос системи (Поліна,
  Олена, Світлана тощо). Якщо українського нема — перейде на російський, далі
  на будь-який доступний.
- Кнопка 🔇 у правому верхньому куті вмикає/вимикає весь звук та озвучку.

## Картинки
- Для іконок використовується Twemoji (CC-BY 4.0) з jsDelivr CDN. Якщо CDN
  недоступний, іконки автоматично замінюються звичайними emoji.
- Поні Пінкі рендериться з SVG-файлів у `assets/pinkie/`.

## Запуск
Відкрий `index.html` у браузері. Можна через простий http-сервер:
```
python3 -m http.server 8000
```
