# Волшебные Пони

Игра про пони для детей 4–8 лет. Работает в любом современном браузере — откройте `index.html`.

## Что внутри
- **Уход за пони** — кормим, моем, расчёсываем, обнимаем, укладываем спать. У пони есть шкалы сытости, чистоты, бодрости и радости. Если давно не приходить — пони начинает скучать.
- **Раскраска** — раскрашиваем единорожку красками из палитры. Нажми на часть пони, чтобы залить её цветом.
- **Аркады** (можно выбрать любую):
  - **Лови звёздочки** — собирай падающие звёзды и яблочки.
  - **Скачи по радуге** — раннер с прыжками через камушки.
  - **Найди пару** — карточки с пони, нужно открывать парами.
- **Уроки доброты** — короткие ситуации с выбором правильного поступка.

## Прогресс
Весь прогресс сохраняется автоматически в `localStorage`:
- состояние пони и время последнего визита (пока тебя нет — пони чуть проголодается),
- цвета раскраски,
- лучшие результаты в каждой аркаде,
- статистика по урокам доброты.

## Картинки
Для иконок используется Twemoji (CC-BY 4.0) с jsDelivr CDN. Если CDN недоступен, иконки автоматически заменяются обычными emoji.

## Запуск
Открой `index.html` в браузере. Можно через простой http-сервер:
```
python3 -m http.server 8000
```
